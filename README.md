# Product Recommendation

This is a service for recommending relevant products for users based on their past behavior and preferences.

It is trained using [Retailrocket recommender system dataset](https://www.kaggle.com/datasets/retailrocket/ecommerce-dataset) available in the Kaggle platform.

The main purpose of this repository is for evaluation towards the second challenge in the Fiap PosTech postgraduate course in [Machine Learning Engineering](https://postech.fiap.com.br/curso/machine-learning-engineering).

**Contents**

These are the most relevant contents found in this project:

1. `notebooks` contain an exploration data analysis of the dataset with insights and decision.
2. `docs` contais the Machine Learning Canvas, the Model Card, and helpful theoretical resources.
3. `data` contains DVC related artifacts that possibility data versioning.
4. `src` contains the the necessary code for training and serving the model for real time inference.
5. `scripts` contains routines for cleaning and preparing data, initialization of infrastructure resources, etc.
